Big Data Assignment 2

Decision Tree

Included files: 
- DecisionTreeMain.java - main run file
- FileReader.java - Class reading in files to list from .csv
- InformationGain.java - Class calculating informatin gain, probability,
			and entropy.
- Node.java - Class node structure for trees.
- Record.java - Class structure for storing data.
- Test.java - Class for testing new test data against built tree.
- Tree.java - Class for building and creating tree.

To run:
1. Change the filepath for "data.csv" in DecisionTreeMain to correct train data path.
2. Change the filepath for "test.csv" in DecisionTreeMain to correct test data path.
3. Run DecisionTreeMain.java
